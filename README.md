Este projeto consiste em um sistema acadêmico desenvolvido em Python, utilizando a biblioteca Streamlit para a interface web. Para que o sistema funcione corretamente, é necessário ter o Python instalado na máquina, preferencialmente na versão 3.10 ou superior, além do gerenciador de pacotes pip, que normalmente já vem junto com o Python. O principal requisito externo do projeto é a biblioteca Streamlit, responsável por toda a interface gráfica da aplicação. Após garantir que o Python está instalado, basta instalar o Streamlit utilizando o comando “pip install streamlit” no terminal ou prompt de comando. Com a biblioteca instalada, o sistema pode ser executado acessando a pasta raiz do projeto e rodando o comando “streamlit run views/app_view.py”, que iniciará o servidor local da aplicação.
